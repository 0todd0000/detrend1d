Metadata¶

The imported NPZ files contain the following metadata variables:

  • cond : (nsteps,) array of integers specifying condition
  • tsteps : (nsteps,) array of floats specifying within-session step times
  • sess : (nsess,) list of session labels
  • tsess : (nsess,) list of session starting times
In [1]:
import os
import numpy as np
import matplotlib.pyplot as plt
import detrend1d as dtr


subj     = 1
fnameNPZ = os.path.join(dtr.dirDATA, f's{subj:03}.npz')
with np.load(fnameNPZ) as z:
    cond   = z['cond']
    tsteps = z['tsteps']
    sess   = z['sess']
    tsess  = z['tsess']

This Metadata class can be used to conveniently perform various operations on the metadata.

The Metadata class contains the following properties / attributes:

nsess     # [int]  total number of sessions
nsteps    # [int]  total number of steps (i.e, observations) across all sessions

snsteps   # (nsess,) [int]  number of observations by session
scondstr  # (nsess,) [str]  condition string labels by session  (reduced labels)
scond     # (nsess,) [int]  condition integer labels by session (reduced labels)
st        # (nsess,) [float]  session starting times (s)

condstr   # (nsteps,) [str]  condition string labels   (reduced labels)
cond      # (nsteps,) [int]  condition integer labels  (reduced labels)
t         # (nsteps,) [float]  step times (seconds)
tr        # (nsteps,) [float]  step times relative to session start (seconds)
ts        # (nsteps,) [float]  session start times (seconds)
